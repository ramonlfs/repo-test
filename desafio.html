Aula 1 - Inspecionar Elementos

Voltar ao topo

É uma Ferramenta com vários tipos de recursos para entendermos mais como determinada página foi criada. Ao pressionar a tecla F12 do teclado, abrirá a ferramente e na aba Elementos, é possível visualizar o HTML da página em questão. Quando quisermos verificar como determinado ponto da tela foi criado, basta clicar em "Selecionar um elemento na página para ser inspecionado" (canto superior esquerdo da tela) ou pressionar o atalho Ctrl + Shift + C. Basta passar o mouse por cima do elemento desejado e na tela de inspeção irá mostrar o código usado para gerá-lo.

Aula 2 - Estrutura Básicado HTML

Voltar ao topo

Sempre que iniciar um projeto, é recomendável que abra uma pasta com o nome do seu Projeto e o seu primeiro, ou seja, a primeira página tenha o nome de "index.html", não é uma obrigação, mas por tradiçao do mundo da Programação, os desenvolvedres aderiram a essa nomeclatura para generalizar o processo.

O HTML consistem em:

1 - TAGs:

Toda TAG no HTML e formada pelo sinal de menor "<", uma palavra "xxx" e um sinal de maior ">", como no exemplo: type="texto" < html> - essa é uma tag para informar ao navegar que tudo que estiver sendo escrito após ela, é referente ao mundo HTML. As Tags abrem e fecham o comando. Usando o mesmo exemplo, usamos a tag para abrir a linha de comando fechamdos acrescentando o sinal de barra "/" após sinal de menos, conforme abaixo:

< html> ... < /html>.

2 - Cabeçalho:

O Cabeçaho é tudo aquilo que irá carregar antes de exibrir para o usuário e composto pela tag < head> ... < /head>.

< html> -Abertura do Mundo HTML

< head> < / head> -Cabeçalho

< /html> -Fechamento do Mundo HTML

3 - Corpo:

O Corpo é toda informaçao que será apresentada para o usuário e é composto pela tag < body> ... < /body>

< html> -Abertura do Mundo HTML

< head> < /head>-Cabeçalho
< body>-Abertura do Corpo da Página
< /body> -Fechamento do Corpo da Página

< /html> -Fechamento do Mundo HTML

Detro do Cabeçalho < head>, existe uma tag chama < title> < /title>, que é o título da página, conforme exemplo abaixo:

< html> -Abertura do Mundo HTML

< head>
< title> Meu Primeiro HTML < /title>
< /head>
< body>-Abertura do Corpo da Página
< /body> -Fechamento do Corpo da Página

< /html> -Fechamento do Mundo HTML

Aula 3 - Falando de TAGs

Voltar ao topo

Existem vários tipos de Tags, cada uma com sua própria função. Abaixo segue alguns exemplos de tags.

< i> < /i> - deixa a letra em Itálico

< strong> < /strong> - deixa a letra em Negrito

< sup> < /su> - para colcar a letra na parte superior da linha, semelhante ao expoenta, na potência matemática.

< u> < /u> - sublinhar o texto

Algumas tags não possuem a Tag de fechamento, ou seja, ela por si só, já um comando.

< input ... /> - gera um campo para o usuário digitar algo no navegador.

< img ... /> - inserir uma imagem na sua página

Aula 4 - Atributos das Tags

Voltar ao topo

Atributos são propriedades que cada tag pode ter. Alguns atributos são mais gertais, ou seja, nem todas as tags podem ter e outras que são específicas.

As Genérias:

id="..." é um tag referete a indicador e serve parea indicar qual elemento o usuário quer manipular.

style= "..." é usado para formatar algum elemento no HMTL e CSS

class="..." é usado para classificar um determinado elemtno. Quando o elemento é classificado, todo aquele, que estiver com a mesma classificação será alterado.

As específicas:

< input ... /> - gera um campo para o usuário digitar algo no navegador.

type="..." - é usado para especificar o tipo de conteúdo que aparecerá ou deve ser preenchido pelo usuário no navegador

< img ... /> - é uma tag que serve para gerar uma imagem e ela precisa da tag src=" endereço da imagem'

< img src="endereço da imagem a ser inserida na página." />

width - serve para alterar a largura da imagem.

Aula 6 - Listas Ordenadas e não Ordenadas

Voltar ao topo

As listas são recursos que serve para apresentar os tópicos de um texto. Elas podem ser divididas em Ordenadas e Não Ordenadas. Nas listas Ordendas os tópicos são apresentadas através de uma sequência numérica, enquando as Não Ordenadas, por pontos. As listas são represtadas por tags que percisam outra tags para sua exibição no navegador.

Ordenadas

< ol> < /ol> - Order list, também chamada de Tag pai.

< li> < /li> - List Item, tamém chamada de Tag filha.

< html>

< head>< title> Meu Primeiro HTML < /title> < /head>
< body>
< ol> Tag Pai
< li>... < /li> - Tag Filha
< li>... < /li> - Tag Filha
< li>... < /li> - Tag Filha
< /ol>
< /body>

< /html>

Não Ordenadas

< ul> < /ul> - Unorder list, também chamada de Tag pai.

< li> < /li> - List Item, tamém chamada de Tag filha.

< html>

< head>< title> Meu Primeiro HTML < /title> < /head>
< body>
< ul> Tag Pai
< li>... < /li> - Tag Filha
< li>... < /li> - Tag Filha
< li>... < /li> - Tag Filha
< /ul>
< /body>

< /html>

Aula 7 - Links

Voltar ao topo

É uma ferramente que serve para que seja possíve navegar no site, ou seja, ao clicar em algum elemento a página seja direcionada a outro ponto da tela, assim como també nos direcionar a outra página da web.

A tag utilizada é a < a ...> - anchor, que quer dizer âncora no Português.

Dentro da Tag, vai ser inserida outra tag "href", que serve para referenciar o local em qur o link vai direcionar. No exemplo abaixo é o site da Dio.me.

Existe uma opção de redirecionar o site para outra página ou para a mesma.

A tag tagert é a resposável por definir em qual aba será aberto o link.

Para abrir na mesma aba, é usada taget="self", já para usar em outra página, é usada target="blank".

É possível também adcionar um balãozinho em cima do elemeto que está ancorado

< html>

< head>< title> Meu Primeiro HTML < /title> < /head>
< body>
< ul>
< li>
< a href="http://dio.me" title=" Clique aqui! " target="self">Dio.me< /a> - para abrir na mesa aba.
< a href="http://dio.me" target="blank">Dio.me< /a> -para abrir em aba diferente.
< /li>
< /ul>
< /body>

< /html>

Em outro arquivo já criado, tambem podemos fazer um link para retornar, referenciando a pirmiera páina.

< html>

< head>< title> Meu Primeiro HTML < /title> < /head>
< body>
< ul>
< li>
< a href="Meu primeiro HTML" title=" Clique aqui!" target="self">< Voltar < /a> - para abrir na mesa aba.
< /li>
< /ul>
< /body>

< /html>
